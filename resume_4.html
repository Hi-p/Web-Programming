■ Java 그림판 애플리케이션 개발 (2025)

Java Swing과 AWT를 활용하여 완전한 기능을 갖춘 그림판 프로그램을 개발하였습니다.
자유 곡선, 직선, 사각형, 원형 등 다양한 도형 그리기 기능과 색상 선택, 선 굵기 조절 기능을 구현하였습니다.
특히 대칭 모드를 통한 좌우 대칭 그리기 기능과 BufferedImage를 활용한 PNG 파일 저장/불러오기 기능을 구현하였습니다.
Graphics2D를 활용한 안티앨리어싱 처리로 부드러운 선 표현을 구현하고, 이벤트 리스너를 통한 사용자 인터페이스를 설계하였습니다.

■ 국내 대기업 소개 웹사이트 제작 (2025)

소프트웨어 프로젝트 과목에서 HTML, CSS를 활용하여 한국의 주요 대기업을 소개하는 웹사이트를 제작하였습니다.
삼성전자, SK, 현대자동차, LG, 포스코, 롯데, 한화, KIA, 네이버, 카카오 등 10개 대기업의 정보를 카드 형식으로 구성하였습니다.
로그인 화면과 반응형 그리드 레이아웃을 구현하여 사용자 친화적인 UI/UX를 제공하였으며, 각 기업의 공식 웹사이트로 연결되는 링크를 추가하였습니다.
CSS 트랜지션과 호버 효과를 활용하여 인터랙티브한 사용자 경험을 구현하였습니다.

■ NewSic 음악 추천 웹사이트 프로젝트 (2025, 팀 프로젝트)

팀원들과 협력하여 음악 입문자를 위한 장르별 아티스트 추천 웹 플랫폼 'NewSic'을 개발 중입니다.
각 팀원이 장르를 분담하여 전문 콘텐츠를 제작하고 있으며, 저는 J-POP 장르를 담당하였습니다.
J-POP 섹션에서 4명의 인기 아티스트 소개 페이지를 구현하였습니다.
HTML5 audio 태그를 활용한 음악 재생 기능과 JavaScript로 앨범 커버 클릭 시 회전 애니메이션 및 음악 재생/일시정지를 구현하였습니다.
CSS3 애니메이션으로 벚꽃 애니메이션 배경과 반응형 카드 레이아웃을 디자인하여 사용자 경험을 향상시켰습니다.
아티스트 간 페이지 네비게이션 기능을 추가하여 사용자의 탐색 편의성을 높였습니다.
